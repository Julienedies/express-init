#项目简介

#项目目录结构

<pre>
├── app                 // 服务器端开发目录
│    ├── app.js         // 应用启动文件
│    ├── conf           // 配置文件目录
│    ├── action         // 路由处理器目录
│    ├── helper         // 自定义辅助类库
│    ├── filter         // 过滤器目录
│    ├── services       // 独立的公共服务目录
│    ├── viewModel      // 视图模型目录
│    ├── model          // 数据层目录
│    ├── test           // 单元测试目录
│    ├── static         // 静态视图目录
│    └── view           // 视图层目录
├── assets              // 静态资源文件目录
│    ├── img            // 图片目录
│    ├── js             // 前端js目录
│    │   └──vendor      // 第三方js库目录
│    └── css            // 样式表目录
├── log                 // 日志目录
├── static              // 前端开发目录
</pre>
